# IBM Data Engineering Lab Work

This repository contains practical code, scripts, and database models developed during the **IBM Data Engineering Professional Certificate** program.

## Skills & Technologies Demonstrated
- **Languages:** Python, SQL
- **Databases:** Relational Databases (RDBMS)
- **Data Engineering:** Automated ETL Pipelines, Data Validation, Data Normalization

## Repository Structure & Key Projects

### [03-Python-Project-for-Data-Engineering](./03-Python-Project-for-Data-Engineering/)
- **Objective:** Architected a Python-based script to extract, transform, and load (ETL) data from unstructured/semi-structured formats.
- **Key Deliverables:** Clean, modularized Python script handling data validation and automated staging.

### [05-Databases-and-SQL-for-Data-Science](./05-Databases-and-SQL-for-Data-Science/)
- **Objective:** Advanced query building, database schema profiling, and multi-table joins.
- **Key Deliverables:** Optimized SQL scripts for cross-industry KPI analysis.
